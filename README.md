# Insurance Cost Prediction

## Project Overview
This project predicts medical insurance charges based on user details 
such as age, BMI, number of children, smoking status, and region.

The model is built using Machine Learning (Linear Regression). 

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Flask

---

## How It Works
1. Load dataset
2. Data preprocessing
3. Convert categorical values to numeric
4. Train Linear Regression model
5. Predict insurance charges
